# Vue 3 + Vite + Bootstrap Login (ESM-compatible)

## Como usar

1. Instale dependências:
   ```
   npm install
   ```

2. Rodar em dev:
   ```
   npm run dev
   ```

3. Build para produção:
   ```
   npm run build
   ```

## Observações

- Editar `src/api/api.js` para apontar `baseURL` para sua API Django no PythonAnywhere.
- Projeto é ESM-only (package.json tem "type": "module") para ser compatível com Node v24.
